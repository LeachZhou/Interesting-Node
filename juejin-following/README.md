# 🐢 juejin-following

## 介绍

- juejin-following是一个批量关注掘金用户爬虫，使用Node.js开发。

## 开发环境

- Node.js@8.11.1

## 安装

```
$ cd juejin-following
$ npm i
$ npm start
```

## 遇到问题

- 如果遇到IP限制问题，可以使用代理IP发送请求 [代理案例1](https://segmentfault.com/q/1010000008196143) [代理案例2](https://blog.csdn.net/luzhongk/article/details/78772286)
